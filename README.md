# Assmira Shop

## Запуск

1. Скопируйте `.env.example` в `.env`.
2. Укажите `TELEGRAM_BOT_TOKEN` и `TELEGRAM_CHAT_ID`.
3. Запустите сайт командой:

```bash
npm start
```

После запуска сайт будет доступен по адресу `http://127.0.0.1:3000`.

## Что защищено

- токен Telegram хранится только на сервере;
- заявки принимаются только с этого же сайта;
- включены защитные HTTP-заголовки;
- есть ограничение частоты заявок;
- данные заказа проверяются на сервере;
- цены пересчитываются на сервере по `catalog.json`;
- добавлено скрытое антиспам-поле.
